# Full-Stack Devops Lab

This repository is a dedicated space for mastering the transition from software development to automated infrastructure. It features a Node.js application that is fully containerized and orchestrated using modern DevOps tools.

## Project Overview
The goal of this project is to demonstrate a complete development-to-deployment workflow.

## Tech Stack
- **Backend:** Node.js
- **Containerization:** Docker
- **Orchestration:** Docker Compose
- **Environment:** Linux (Ubuntu/CentOS)
- **Version Control:** Git
